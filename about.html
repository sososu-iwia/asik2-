About this mini‑site

This project is a simple yet functional 4-page website built entirely with HTML and CSS. It features a homepage, a catalog of shows, a sample detail page, and an About page. All placeholder texts and images can be replaced with custom content, allowing the site to serve either as a personal portfolio or as a class assignment. The website was thoughtfully designed and developed by Damir and Ali, who worked together to combine creativity and technical skills in order to produce a clean, user-friendly, and visually appealing result.

How to customize

  1. Open assets/css/main.css to tweak colors, spacing, and layout.
  2. Edit each .html file to update copy and links.
  3. Replace images in assets/images/ with your posters or photos (keep the same names or change the src).
  4. Duplicate stranger-things.html to create more detail pages.
